# AI-103 Course Orientation

> [!quote]
> A one-page map of **Exam AI-103: Develop AI apps and agents on Azure** — the exam, the five
> skill domains, the five hands-on lab repos, and the vocabulary the current exam grades on.
> Every Azure claim links to Microsoft Learn so you can re-check it as the surface keeps changing.

This is a study aid, not official Microsoft courseware. The authoritative sources are
always the linked Learn pages.

## What AI-103 is

Passing the exam earns the **[Microsoft Certified: Azure AI Apps and Agents Developer Associate](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-apps-and-agents-developer-associate/)**
credential. It validates building, managing, and deploying AI solutions on **[Microsoft Foundry](https://learn.microsoft.com/en-us/azure/ai-foundry/)** using **Python**.

| | |
|---|---|
| **Exam** | AI-103 |
| **Certification** | Azure AI Apps and Agents Developer Associate (Associate, Intermediate) |
| **Instructor-led course** | [AI-103T00-A: Develop AI apps and agents on Azure](https://learn.microsoft.com/en-us/training/courses/ai-103t00) (4 days) |
| **Length / format** | 120 minutes, proctored, may include interactive (lab) components |
| **Prerequisites** | Comfortable with Python; familiar with using APIs/SDKs and core Azure services |
| **Official study guide** | [aka.ms/AI103-StudyGuide](https://aka.ms/AI103-StudyGuide) |

## The five skill domains

These are the functional groups **assessed on the exam**, verbatim from the certification page. (The
per-domain weighting percentages live in the official [study guide](https://aka.ms/AI103-StudyGuide) —
check it for the current split, since Microsoft revises the weights periodically.)

1. **Plan and manage an Azure AI solution**
2. **Implement generative AI and agentic solutions**
3. **Implement computer vision solutions**
4. **Implement text analysis solutions**
5. **Implement information extraction solutions**

## The five hands-on lab repos

The course draws its labs from **five** Microsoft Learning repositories. Each has a rendered
instructions site (what you follow in class) built from the repo (what you clone).

| Skill module | GitHub repo | Lab instructions site |
|---|---|---|
| AI-3016 · Generative AI apps | [mslearn-ai-studio](https://github.com/MicrosoftLearning/mslearn-ai-studio) | [instructions](https://microsoftlearning.github.io/mslearn-ai-studio/) |
| AI-3026 · AI agents | [mslearn-ai-agents](https://github.com/MicrosoftLearning/mslearn-ai-agents) | [instructions](https://microsoftlearning.github.io/mslearn-ai-agents/) |
| AI-3003 · Natural language solutions | [mslearn-ai-language](https://github.com/MicrosoftLearning/mslearn-ai-language) | [instructions](https://microsoftlearning.github.io/mslearn-ai-language/) |
| AI-3008 · Computer vision solutions | [mslearn-ai-vision](https://github.com/MicrosoftLearning/mslearn-ai-vision) | [instructions](https://microsoftlearning.github.io/mslearn-ai-vision/) |
| AI-3022 · Information extraction solutions | [mslearn-ai-information-extraction](https://github.com/MicrosoftLearning/mslearn-ai-information-extraction) | [instructions](https://microsoftlearning.github.io/mslearn-ai-information-extraction/) |

> [!info] There is no single "AI-103" repo — and that trips people up
> Microsoft organizes lab content **by skill module (`AI-30xx`), not by course**. Each
> `mslearn-ai-*` repo is a standalone bundle reused across several courses and self-paced Learn
> paths; AI-103 simply pulls labs from all five. There is **no aggregator repo** in the
> [MicrosoftLearning org](https://github.com/orgs/MicrosoftLearning/repositories). The real upstream
> hub is the Learn **course page** above — this table is the closest thing to a one-page index of
> every repo for this specific course.

## Vocabulary the current exam grades on

The Azure AI surface was renamed and reorganized recently. Model-trained tools and older blog posts
still teach the obsolete names — and the exam grades on the **current** ones. When in doubt, trust
the linked Learn page over any table (including this one).

| Use this | Not this (older / obsolete) |
|---|---|
| [Microsoft Foundry](https://learn.microsoft.com/en-us/azure/ai-foundry/) | Azure AI Foundry, Azure AI Studio |
| Foundry Tools | Azure AI services, Cognitive Services |
| [Foundry Agent Service](https://learn.microsoft.com/en-us/azure/ai-foundry/agents/) | Azure AI Agent Service |
| [Content Understanding](https://learn.microsoft.com/en-us/azure/ai-services/content-understanding/) | (new — multimodal extraction across documents, images, audio, video) |
